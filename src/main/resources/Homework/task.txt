Задача для самостоятельного решения.
Дано:
    XML-сообщение, которое вы планируете отправить какому-то сервису. В данном сообщении, значения для некоторых
    полей заданы шаблоном ${имя_переменной}. Также присутствует XSD-схема, которая валидирует данное сообщение.
    На текущий момент показывает что шаблон невалиден.

Что требуется:
    Программно сгенерировать XML-файл по предложенному шаблону (заменить в исходном XML все шаблонные части конкретными
    значениями, применяя поиск по XPath).
    Программно провалидировать полученный файл.
